=================
 UnTinyURL 0.9.1
=================

UnTinyURLとは
=============
UnTinyURLは、TinyURLなどで短縮されたリンクの元URLを表示する、Safari用プラグインです。

動作要件
========
* Mac OS X 10.5 以降
* Safari 3.0 以降
* SIMBL がインストール済みであること

インストール
============
UnTinyURL.bundleを以下のいずれかのフォルダに移動してください。

* /Library/Application Support/SIMBL/Plugins/
* ~/Library/Application Support/SIMBL/Plugins/

アンインストール
================
UnTinyURL.bundleを上記のフォルダから削除してください。

使い方
======
TinyURLなどで短縮されたURLへのリンクにカーソルを合わせてください。ツールチップに元のURLが表示されます。
対応しているURL短縮サービスは以下の通りです。

* TinyURL ( http://tinyurl.com/ )
* bit.ly  ( http://bit.ly/ )
* is.gd   ( http://is.gd/ )
